Questions below were designed to showcase the different features of pl_checkbox. The goal of pl_checkbox is to allow users to select multiple options from a list of choices. To restrict choice to only one option, e.g. what comes from a standard multiple choice, please consider using pl_multiple_choice.

Within this question, you can observe the default behavior of the pl_checkbox element. In particular, all options are shown after being randomized with help text at the bottom to remind users to select all possible options.

Group work provides which of the following benefits:

Exposure to new viewpoints from different fields and life experiences. Improved creativity and overall work quality. The ability to work on a project by yourself. Having only one person make all of the decisions. Constructive dialog and increased internal group motivation. Personal accountability to the team to work on a project. Friendship. The resources to work on a small project instead of a large project. Prepares you for a team-environment in the workplace or a research group.

In this variant, the pl_checkbox element is restrict to displaying no more than 5 choices with between 2 to 4 correct answers. Information regarding the range of correct answers users should select can be displayed by setting detailed_help_text="true"
