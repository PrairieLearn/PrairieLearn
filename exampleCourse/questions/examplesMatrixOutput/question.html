The questions below were designed to showcase the different features of pl-matrix-output. The goal of pl-matrix-output is to allow users to easily copy parameters (scalar values or 2D matrices) for various computational tools.

Within this question, you can observe the default behavior of the pl-matrix-output element. In particular, the default matrix output has Matlab, Mathematica, and Python tabs. The default active tab is Matlab, this can be changed using the attribute default-tab. This element has the ability to display any 2D matrix or scalar value and they are displayed with 2 digits after the decimal.

This is very similar to Matlab syntax, but it takes into account Mathematica reserved variables (C, D, E, I, K, N, O) and formats comments correctly. The reserved variables are appended with an 'm' in the Mathematica tab only. This uses the optional parameter show-mathematica.
