Questions below were designed to showcase the different features of pl-matrix-output. The goal of pl-matrix-output is to allow users to easily copy values for various computational tools.

Within this question, you can observe the default behavior of the pl-matrix-output element. In particular, the default matrix output has a Matlab and a Python tabs and the parameters are displayed with 3 digits. This element has the ability to display any matrix or scalar value.
