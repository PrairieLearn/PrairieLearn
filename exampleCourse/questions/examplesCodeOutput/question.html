The following examples below were created to show the capabilities of the pl-code-output element, which allows formatting and rendering various Python variables defined in the server.py on a question page. For example, Pandas DataFrames can be set to a parameter and then rendered on the page a la Jupyter Notebook, or ordinary Python variables can be displayed in text format.
