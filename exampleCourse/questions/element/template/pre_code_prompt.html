Prompt

Explain, in English, how to solve the problem given in the Programing Problem section. When writing your explanation

  1. Use Markdown. You can find a cheat sheet containing all the Markdown syntax at https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet
  2. Use the format we used when solving the Tic-Tac-Toe problem in class That is you should
    • Start from the top of the problem and work your way down
    • Have a level two heading (## My Heading) for each major part of your problem
    • You should have a numbered list for each step in your process
    • Steps that should be repeated or are conditional should be indented one level inside the repetition/conditional condition
  3. You cannot mention any coding constructs in your solution. This includes but is not limited to
    • Types (int, float, string, array, etc.)
    • Functions (printf, scanf, pow, atoi, etc.)
    • If statements
    • Loops
  4. Your explanation should be understandable by someone with NO prior programming experience

Grading

We will be looking for two things when grading your assignment

  1. You made a sincere, honest attempt to explain how to solve the problem. Even if your solution is incorrect, as long as your solution looks reasonable, you will be given credit.
  2. You don't have any code related constructs in your solution. Every mention of one will result in a {{code_penalty}}{{^code_penalty}}10{{/code_penalty}}% penalty.
