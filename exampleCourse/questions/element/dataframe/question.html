The following examples below were created to show the capabilities of the pl-dataframe element, which allows formatting and rendering of pandas DataFrames, e.g. pd.DataFrame(), inside the server.py to be displayed on the question page.

Here is an example with a Pandas DataFrame input. Note that if you are setting a value in data["params"] to a DataFrame, you can use PrairieLearn's built-in pl.to_json() function. We've set show-dtype="true" and display-language="python"

By default, a Pandas DataFrame will display its index column and header row, as well as a postscript label with the table dimensions. You can toggle these with attributes on the pl-dataframe tag. Here, we've set the following: show-header="false" show-index="false" show-dimensions="false" show-dtype="none" digits="4" display-variable-name="data_frame" width="10_000":

Here's another Pandas DataFrame. This DataFrame has timestamps and uses the new encoding rule in pl.to_json() by setting the keyword argument df_encoding_version=2. We also set show-python="false" to hide the tab showing code for reproducing the dataframe.
