The following examples below were created to show the capabilities of the pl-dataframe element, which allows formatting and rendering of pandas data frames, e.g. pd.DataFrame(), inside the server.py to be displayed on the question page.

Here is an example with a Pandas DataFrame input. Note that if you are setting a value in data["params"] to a DataFrame, you can use PrairieLearn's built-in pl.to_json() function. We've set show-dtype="true"

By default, a Pandas DataFrame will display its index column and header row, as well as a postscript label with the table dimensions. You can toggle these with attributes on the pl-dataframe tag.
