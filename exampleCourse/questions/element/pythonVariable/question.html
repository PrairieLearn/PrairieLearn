The following examples below were created to show the capabilities of the pl-python-variable element, which allows formatting and rendering various Python variables defined in the server.py on a question page. The semantics of this element are very similar to those of the Python pprint library, and we encourage reviewing documentation for that library for more detailed examples.

These examples are drawn from the pprint library documentation and a tutorial for using this library.
