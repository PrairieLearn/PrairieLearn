Questions below were designed to showcase the different features of pl-code. The goal of pl-code is to display different programming languages with appropriate syntax highlighting and code line emphasis.

The most common use of pl-code is to display pre-written code with appropriate syntax highlighting. The language must be specified with the language attribute. In this case, we are requesting python with language="python". We also enable the copy code button by setting copy-code-button="True". The pl-code element can also be used to show output of a terminal command using ANSI colors, by setting the language attribute to ansi-color.
