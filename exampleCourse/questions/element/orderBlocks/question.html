This question illustrates the basic use of pl-order-blocks.

Drag all the prime numbers below to the answer zone. Remember 1 is not a prime number.

This example uses the attribute source-blocks-order="ordered" so that the displayed blocks appear in the same order as defined in the HTML (rather than the default "alphabetized" which would sort them alphabetically). It also uses the attribute grading-method="unordered", so that any ordering of the solution is accepted as correct.
