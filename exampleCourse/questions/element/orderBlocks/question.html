This question illustrates different use cases for the pl-order-blocks.

Example 1: Drag all the prime numbers below to the yellow box. Remember 1 is not a prime number.

This example uses the attribute source-blocks-order="ordered" so that the displayed blocks appear in the same order provided by the HTML answers. It also uses the attribute grading-method="unordered", so that any ordering of the solution is accepted as correct.

Example 2: Drag all the prime numbers below to the yellow box in ascending order.

This example uses the attributes min-incorrect="2" and max-incorrect="4" to display a minimum of 2 incorrect distractors and maximum of 4. Since this example uses the default values for source-blocks-order="random" and grading-method="ordered", the blocks in the source area appear shuffled, and the correct answer from the solution area follows the order defined by the pl-answers.

Example 3: Implement the Python function my_sum, which takes 2 numbers, first and second, and return their sum. Note that the ordering of instantiation, computation, and returning of the value must be correct. Remember to indent your code!

This examples uses grading-method="ranking" to control how the correct solution is ordered and indent attribute in pl-answer to define expected code indentation.

def my_sum(first, second): sum = 0 sum += first sum += second return sum sum + first sum + second This is an example very long tile to test CSS overflow

Example 4: Let $a, b$ be any even numbers. Prove that $a + b$ is always even.

This example uses the attribute grading-method="ranking" to control how the correct solution is ordered. When the order of a correct block is not relevant, the attribute ranking is not provided. In this example, two of the blocks can be placed at any position when constructing the solution. The attribute solution-placement="bottom" is used to display the input area above the output area.

 Proof by Contrapositive: Take some $m \in \mathbb{Z}$ Take some $n \in \mathbb{Z}$ Assume $m$ is even Assume $n$ is even $\exists \ell \in \mathbb{Z}$ such that $m = 2l$ $\exists k \in \mathbb{Z}$ such that $n = 2k$ $am^2 + bm + cn^2 + dn + e$ $= a(2l)^2 + b(2l) + c(2k)^2 + d(2k) + e$ $= 2(2al^2 + bl + 2ck^2 + dk) + e$ Since $e$ is odd, $2(2al^2 + bl + 2ck^2 + dk) + e$ is odd $QED$
