This question illustrates different use cases for the pl-order-blocks.

Example 1: Drag all the prime numbers below to the answer zone. Remember 1 is not a prime number.

This example uses the attribute source-blocks-order="ordered" so that the displayed blocks appear in the same order provided by the HTML answers. It also uses the attribute grading-method="unordered", so that any ordering of the solution is accepted as correct.

Example 2: Drag all the prime numbers below to the answer zone in ascending order.

This example uses the attributes min-incorrect="2" and max-incorrect="4" to display a minimum of 2 incorrect distractors and maximum of 4. Since this example uses the default values for source-blocks-order="random" and grading-method="ordered", the blocks in the source area appear shuffled, and the correct answer from the solution area follows the order defined by the pl-answers.

Example 3: Implement the Python function my_sum, which takes 2 numbers, first and second, and return their sum. Note that the ordering of instantiation, computation, and returning of the value must be correct. Remember to indent your code!

This examples uses grading-method="ranking" to control how the correct solution is ordered and indent attribute in pl-answer to define expected code indentation. This example also demonstrates the use of the attribute distractor-for to visual pair together blocks with their distractor blocks.

def my_sum(first, second): sum = 0 sum += first sum += second return sum sum + first sum &= first sum + second

Example 4: Prove that $A \land B \land C \rightarrow A \land C$

This example uses the attribute grading-method="dag" to control how the correct solution is ordered. This is especially useful when you want students to construct a mathematical proof, where the dependency of lines of the proof on one another is given by a directed acyclic graph (DAG) structure. The depends attribute on each line is used to denote which lines of the proof must appear before a particular line. Then a correct answer is a topological sort of the lines of the proof. The attribute solution-placement="bottom" is used to display the input area above the output area.

 Assume $A \land B \land C$. Then $A$ is true. Then $C$ is true. Since $A$ and $C$, we know $A \land C$
