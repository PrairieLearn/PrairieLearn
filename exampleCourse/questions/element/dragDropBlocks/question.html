Example 1: Drag all the prime numbers below to the yellow box. Remember 1 is not a prime number.

Example 2: Drag all the prime numbers below to the yellow box in ascending order.

This example uses the attribute shuffle-options="true" so that the displayed blocks are shuffled in random order. It also uses min-incorrect="2" and max-incorrect="4" to display a minimum of 2 incorrect distractors and maximum of 4.

Example 4: Let $a, b$ be any even numbers. Prove that $a + b$ is always even.

Let $a + b$ be even. Implying $a, b$ are both even. Let $a$ be even. Let $b$ be even. Then $a = 2n, b = 2m$. Implying $a + b = 2 (n+m)$ $QED$
