Questions below were designed to showcase the different features of pl-units-input. The goal of pl-units-input is to allow users to enter real or complex values to be checked within a specific tolerance. The correct value of $c$ in all questions below is {{correct_answers.c_1}}.

Within this question, you can observe the default behavior of the pl-units-input element. In particular, the default input field has help text as a placeholder as well as a question button and is set to use "relabs" to check the tolerance, which is a combination of relative ("rtol") and absolute ("atol") tolerances.
