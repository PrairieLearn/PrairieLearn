This question demonstrates the weight, allow-blank, and show-score options for pl-drawing.

  • weight="3": The drawing is worth 3x the integer input (which defaults to weight 1). If you get only the area correct, you score 25% (1/4). If you get only the centroid correct, you score 75% (3/4).
  • allow-blank="true": Submitting without placing any points will score 0% for the drawing part instead of showing "Invalid submission".
  • show-score="true": After submission, a score badge (green/yellow/red) appears next to the drawing in the "Submitted answer" panel.
