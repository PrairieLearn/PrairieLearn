Questions below were designed to showcase the different features of pl-integer-input. The goal of pl-integer-input is to allow users to enter only integer values. That is, users are only able to enter whole numbers like -10, 0, 3, 44, and so on. The correct value of $c$ in all questions below is {{correct_answers.c_1}}.

Within this question, you can observe the default behavior of the pl-integer-input element. In particular, the default input field has help text of "integer" as a placeholder as well as a question button.
