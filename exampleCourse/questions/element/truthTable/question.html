Questions below were designed to showcase the different features of pl-truth-table. The goal of the element is to allow users to create a truth table with dropdowns to select the correct answer given the different values in an expression.

Part 1

Here, you can observe the default behavior of the pl-truth-table element. The order of the variable names is randomized, which effectively randomizes the order of the problems.

x y X or Y
Part 2

Here, the pl-truth-table element uses Mustache sections to display the variables as well as has multiple answer columns. The first answer column shows that other elements can be used within the pl-answer-column element, such as a pl-code element for syntax highlighting. This first answer column also shows that a pl-answer-column element's expression attribute is optional and the answer column's expression will default in that case to the text that lies within the column. The number of rows that appear is also customized using the num-rows attribute. The number of rows that appears is also limited to three via the num-rows attribute.

{{#params.variables}} {{name}} {{/params.variables}} A or B C and D
Part 3

Here, the true and false labels are set to custom values using the true-label and false-label attributes. The order of the variables is also set to be a fixed order using the fixed-order attribute, effectively making the order of the problems fixed as well.

A B C A or B and C
