Questions below were designed to showcase the different features of [`pl-code`](http://prairielearn.readthedocs.io/en/latest/elements/#pl-code-element). The goal of [`pl-code`](http://prairielearn.readthedocs.io/en/latest/elements/#pl-code-element) is to display different programming languages with appropriate syntax highlighting and code line emphasis.
The most common use of `pl-code` is to display pre-written code with appropriate syntax highlighting. The language must be specified with the `language` attribute. In this case, we are requesting python with `language="python"`. def square(x): return x * x
Another useful feature of `pl-code` is the ability to highlight specific lines. This is done by using `highlight-lines="low-high"` attribute. Moreover, one can disable selecting the code region by adding `prevent-select="true"`. # Here's a version that highlights specific lines of code and disables selection def foo(): return 'foo' def square(x): return x * x def bar(x): return 'bar'
Lastly, `pl-code` also provides the ability to have code included from an external file. This can be achieved with `source-file-name="my_code_file.py"` attribute.
