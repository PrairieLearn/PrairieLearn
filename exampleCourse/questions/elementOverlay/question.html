The element pl-overlay allows the positioning of pre-existing elements in arbitrary positions on top of other elements. This question combines the pl-drawing element to serve as a background, along with the number input element.

Each overlayed element is wrapped with a <pl-location x=... y=...> tag, which specifies the location of the child relative to the parent overlay element. The optional centered="true" attribute can be used to center contents about the specified location.
