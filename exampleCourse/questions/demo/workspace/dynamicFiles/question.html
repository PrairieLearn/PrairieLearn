This is a workspace question with an in-browser terminal (xterm.js) and dynamic files. These files are created using the generate function in server.py.

This question demonstrates:

  • Creating dynamic files with various encodings (text, hex, base64)
  • Including files from the question or course directory
  • Creating executable scripts with the mode parameter (see run_tests.sh)
  • Template files with executable permissions preserved (see template_script.sh)

Try running ./run_tests.sh or ./template_script.sh in the terminal to see the executable files in action.
