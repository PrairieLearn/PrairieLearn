The element pl-overlay allows the positioning of pre-existing elements in arbitrary positions on top of other elements. This question combines the pl-drawing element to serve as a static background (no drag and drop functionality enabled here), along with the number input element.

Each overlayed element is wrapped with a <pl-location x=... y=...> tag, which specifies the location of the child relative to the parent overlay element. The halign and valign attributes can be used to center contents about the specified location.

The optional <pl-background> tag can be used to wrap some contents as the "background", which will automatically resize the overlay to fit the background.
