If an integer is divisible by 4, it must be divisible by 2. If an integer is divisible by 2, it must be divisible by 4. An integer divided by an integer is always an integer. The number 3.5 is an integer.

This example question shows how we can use the Coverage (COV) grading scheme in a multiple-choice question where only one option is correct, but multiple options can be selected. In this question, COV grading gives a nonzero amount of credit to any valid submission that includes the correct option. Notice also that at most 3 options can be selected; we accomplish this by setting max-select="3" in the pl-checkbox attributes.
