You can use Proof Blocks by using a pl-order-blocks element with grading-method="dag". Use the depends attribute on each line is to denote which lines of the proof must appear before that line. The autograder will use this dependency information to construct the dependency graph of the proof. Then a correct answer is a topological sort of the lines of the proof. This example also uses the option feedback="first-wrong" to give the student feedback about where the first wrong line of their proof was.

Example 1: Use a proof by contrapositive to show that if $2m^2 + n + 1$ is even then either $m$ or $n$ are odd.

Proof by Contrapositive: Take some $m \in \mathbb{Z}$ Take some $n \in \mathbb{Z}$ Assume $m$ is even Assume $n$ is even $\exists l \in \mathbb{Z}$ such that $m = 2l$ $\exists k \in \mathbb{Z}$ such that $n = 2k$ $2m^2 + n + 1 = 2(2l)^2 + 2k + 1 = 2(4l^2 + k) + 1$ Then since $2(4l^2 + k)$ is even, $2(4l^2 + k) + 1$ is odd.

Example 2:Let $A, B,$ and $C$ be propositions. Prove that $(A \land C) \lor (A \land B) \rightarrow A $

This example uses the pl-block-group element, which can be used to divide a proof into cases. In a proof that has cases, a correct answer is any topological sort of the lines, with the added condition that any lines in the same pl-block-group element must appear contiguously in the solution, as when you are writing a proof by cases, you finish one case before proving the next case.

Proof by Cases: Case $A \land C$: Since $A \land C$ is true, we know $A$ is true. Case $A \land B$: Since $A \land B$ is true, we know $A$ is true.
