This is a small example question that shows how we can use mustache processing on test files much like question.html. server.py randomizes a function name and a list of input/output pairs the function must correctly handle plus a default output for every other case. The test file checks these pairs and the default output.

To enable mustache processing, you must ensure the environment variable PL_PROCESS_MUSTACHE_TEMPLATES exists with the value true. A good way to do that is to include the following in your info.json file: 

                {
                    "externalGradingOptions": {
                        "image": "prairielearn/grader-python",
                        "environment": {
                            "PL_PROCESS_MUSTACHE_TEMPLATES": "true"
                        }
                    }
                }

In the files, review tests/ans.mustache.py and tests/test.mustache.py to see how mustache processing is being used.
