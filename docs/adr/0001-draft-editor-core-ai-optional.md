# Draft Editing Is Core; AI Assistance Is Optional

Draft questions are a core authoring concept, not an AI-only feature. The Draft editor therefore
belongs in the non-EE application and must support editing, previewing, and finalizing draft
questions without enterprise licensing. AI chat and AI-assisted revision remain optional EE features
inside that editor, so template-based and scratch draft creation work for all PrairieLearn installs
while AI-specific behavior stays feature-gated.
