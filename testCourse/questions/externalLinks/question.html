This question embeds external links in its content so that LockDown Browser's content-link behavior can be exercised. LockDown Browser does not stop a content link from opening — clicking one of the links below opens it in a new window. What is restricted is navigating onward from there: a child window opened from the quiz is allowed only a single navigation, after which further navigation shows "LINK IS BLOCKED" on the toolbar.
