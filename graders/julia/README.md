# `prairielearn/grader-julia`

View documentation at <https://docs.prairielearn.com/julia-grader>.

This image provides the Julia external grader used by PrairieLearn. It follows the same
`/grade/student`, `/grade/tests`, and `/grade/results` contract as the other maintained grader
images, and it supports both a PLTestCase-style authoring flow and a standard Julia `Test`
macro flow.
