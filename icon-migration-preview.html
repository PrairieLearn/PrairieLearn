Font Awesome → Bootstrap Icons Migration

Preview of icon replacements for apps/prairielearn/src/pages and apps/prairielearn/src/components

Special cases requiring manual attention:
  • fa-ul / fa-li — Font Awesome list styling (used in workspace modals). Replace with standard <ul> with custom CSS or Bootstrap list groups.
  • fa-spin / fa-pulse — Animated spinner classes. Use Bootstrap's spinner-border component or CSS animation.
  • fa-10x / fa-2x / fa-2xl — Size modifiers. Use inline font-size style instead.
  • fa-width-auto — Custom width class. Can be dropped or replaced with custom CSS.
