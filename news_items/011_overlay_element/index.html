Thanks to the efforts of Nicolas Nytko (CS BS '19, CS MS '22), the new pl-overlay element allows you to easily place existing elements on top of each other in varying positions. This now enables, for example, questions that require student input on specific parts of diagrams and figures:

Each placed element in the overlay is wrapped with a pl-location tag, which lets you specify positioning and alignment relative to the overlay area. The source to the above example is given here: 

<pl-overlay width="700" height="600" clip="false">
    <pl-location left="0" top="0" valign="top" halign="left">
       <pl-figure file-name="map.png" directory="clientFilesQuestion"></pl-figure>
    </pl-location>
    <pl-location left="460" top="260" valign="top" halign="center">
       <pl-string-input answers-name="string_value" show-help-text="false" size="7"
                        correct-answer="Ukraine" remove-leading-trailing="true"
                        ignore-case="true"></pl-string-input>
    </pl-location>
</pl-overlay>
