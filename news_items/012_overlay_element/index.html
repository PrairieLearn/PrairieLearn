Nicolas Nytko (CS BS '19, CS MS '22) has added an amazingly flexible new pl-overlay element that lets you place elements on top of each other in arbitrary locations. For example, you can put a pl-string-input on top of a pl-figure to ask students to label a diagram:

Each sub-element in the overlay is inside a pl-location tag, which lets you specify the position and alignment relative to the overlay area. The source to the above example is given here: 

<pl-overlay width="700" height="600" clip="false">
    <pl-location left="0" top="0" valign="top" halign="left">
       <pl-figure file-name="map.png" directory="clientFilesQuestion"></pl-figure>
    </pl-location>
    <pl-location left="460" top="260" valign="top" halign="center">
       <pl-string-input answers-name="string_value" show-help-text="false" size="7"
                        correct-answer="Ukraine" remove-leading-trailing="true"
                        ignore-case="true"></pl-string-input>
    </pl-location>
</pl-overlay>

For more information, see the pl-overlay documentation.
