Externally graded questions now have the ability to return a status of invalid or not gradable. This can be used to give feedback to students if their code cannot compile because of a syntax error, for example. Any invalid answers will not use up one of a student's attempts on exams.

To set a submission as invalid from the external grader, set the key "gradable": false in the results.json file. You can also put one or more (or zero) errors to display in format_errors.

A full invalid attempt could look something like: 

{
    "gradable": false,
    "format_errors": ["Error 1", "Error 2", "Error 3"],
    "message": "Here is a more detailed description of why your answer is incorrect."
}
