After extensive feedback, Jonatan Schroeder, with input from James Yu (University of British Columbia), has improved the user interface for two major pages in the PrairieLearn instructor interface: the instructor Gradebook page, and the Students page for an assessment.

New features include:

  • Headers remain visible even when scrolling down, simplifying the identification of which values refer to each column;
  • A built-in search function allows instructors to filter specific students;
  • Instructors can select which columns are visible or hidden, allowing for easier analysis and comparison of grades between assessments;
  • Some updates like grade changes and time limit updates are performed without reloading the page, maintaining sort, search, scrolling and column selection intact;
  • The assessment students page can be refreshed while the assessment is active without reloading the page, including an option to auto-refresh every 30 seconds;
  • The assessment students page also includes pagination, which allows instructors of large classes to load the page faster.
