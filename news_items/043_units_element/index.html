PrairieLearn now has a new pl-units-input element, thanks to the efforts of Eliot Robson (UIUC).

This element can be used to create problems where students must input a quantity with an accompanying unit. By default, comparisons to a reference solution can be performed with absolute tolerances in a reference unit.

For more information, see the pl-matching documentation.
